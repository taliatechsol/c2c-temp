## Knowledge Graph Navigation

1. If a knowledge graph index or representation exists (e.g. in `graphify-out/` or `graphify/`), consult the corresponding `GRAPH_REPORT.md` and `graph.json` files to understand relevant boundaries and import/data flows.
2. Refer to the structural map to locate APIs and database schemas before writing new query layers.
